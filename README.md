# To-Do List Mission System - Ola Technologies

# Descripción
Aplicación web full-stack desarrollada en Python/Flask que permite la gestión de tareas mediante una interfaz temática de objetivos de misión.

# Stack Tecnológico
- **Backend:** Python 3.x, Flask.
- **Base de Datos:** SQLite con SQLAlchemy (ORM).
- **Frontend:** HTML5, CSS3 (Bootstrap + Animate.css), JavaScript.

# Instalación
1. Instalar dependencias: `pip install flask flask-sqlalchemy`
2. Ejecutar la aplicación: `python app.py`
3. Abrir en el navegador: `http://127.0.0.1:5000`

# Requerimientos Cumplidos
- Persistencia de datos en SQLite.
- Filtros dinámicos (Todas, Pendientes, Completadas).
- Edición en línea (Toggle Label/Input).
